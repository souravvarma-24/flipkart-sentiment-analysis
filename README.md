# Sentiment Analysis of Flipkart Product Reviews

This project performs sentiment analysis on Flipkart product reviews using machine learning.

## Models Used
- TF-IDF + Logistic Regression
- Bag of Words + Logistic Regression
- Naive Bayes
- Word2Vec
- BERT (Sentence Transformers)

## Web App
- Built using Streamlit
- Accepts real-time user review input
- Predicts sentiment as Positive or Negative

## Deployment
- Streamlit Web Application
- Ready for AWS EC2 deployment

## Evaluation Metric
- F1-Score
